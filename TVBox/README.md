## 配置地址 简易设定流程：
* 设置 --> 设置其他  --> 配置地址 (底下二选一)
	1. 请输入 https://github.com/YuanHsing/freed/raw/master/TVBox/meow.json --> 确认
	2. CH猫友免墙地址 请输入 http://freed.yuanhsing.cf/TVBox/meowcf.json --> 确认

## 切换数据源
* 设置 --> 站点数据源 --> 右侧页面选取任一数据源 --> 设为首页数据源 --> 返回首页

## 失效数据源
* 设置 --> 站点数据源 --> 右侧页面选取失效的数据源 --> 禁用
